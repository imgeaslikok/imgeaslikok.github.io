Multi-Focus Image Fusion

Getting 'all-in-focus' images by using Multi-focus Image Fusion techniques.

  • Researching papers selecting the best approach(Amin-Naji)
  • Collecting same images that has different focus point.
  • Application of low pass filter and shifting.
  • Conversion to DCT domain.
  • Calculation of correlation coefficient.
  • Selecting the image block that is focused.
  • Application and interface development on Matlab and tests.

Cyberbullying Detection

Creating a training model for Cyberbullying Detection using positive and negative Youtube comments as dataset.

  • Tagging the data to be positive if the positive value is 1 and negative if the positive value is 0.
  • Data preprocessing such as tokenizing, removing stopwords, numbers and special characters
  • Creating word vectors by using TF, TF-IDF,Binary Weighting methods
  • Using SVM, Multinomial Naive Bayes and Random Forest classifiers for training.

Monitoring Water Resources

  • To analyze Index Methods(NDWI, MNDWI, NDPI, AWEI), Snake Algorithm, Active Contour.
  • Labeling with water, land, wet zone by our own hands.
  • Testing all methotlads for the Tchad Lake.
  • Writing a paper.
